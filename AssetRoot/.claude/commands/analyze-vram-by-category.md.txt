# analyze-vram-by-category

Launch画面でGPU上の全テクスチャをカテゴリ別（アセット/シャドウマップ/スカイボックス/フォント/その他）に分類し、VRAM使用量を集計・一覧表示する。

## 前提条件
- PlayCanvas EditorからLaunchでアプリケーションを実行中であること

## 手順

### 1. Launch タブに切り替える

```bash
playwright-cli tab-list
playwright-cli tab-select <tab-index>
```

### 2. カテゴリ別VRAM情報を取得

アセット登録テクスチャの判定にはオブジェクト完全一致（Set.has）を使用する。
`graphicsDevice.textures` は Set なので `forEach` でイテレートすること（for ループ不可）。

```bash
playwright-cli eval "() => { var assetTextures = new Set(); pc.app.assets.filter(function(a) { return a.type === 'texture' && a.resource; }).forEach(function(a) { assetTextures.add(a.resource); }); var summary = { asset: 0, shadow: 0, skybox: 0, font: 0, other: 0 }; var details = []; pc.app.graphicsDevice.textures.forEach(function(tex) { var size = tex._gpuSize || tex.gpuSize || 0; if (!size) { var bpp = 4; var faces = tex.cubemap ? 6 : 1; var mips = tex.mipmaps ? 1.33 : 1; size = tex.width * tex.height * bpp * faces * mips; } var name = (tex.name || '').toLowerCase(); var cat = 'other'; if (assetTextures.has(tex)) { cat = 'asset'; } else if (name.indexOf('shadow') !== -1 || name.indexOf('depth') !== -1) { cat = 'shadow'; } else if (name.indexOf('skybox') !== -1 || name.indexOf('prefiltered') !== -1 || name.indexOf('envmap') !== -1 || tex.cubemap) { cat = 'skybox'; } else if (name.indexOf('font') !== -1 || name.indexOf('atlas') !== -1) { cat = 'font'; } summary[cat] += size; details.push({ name: tex.name || '(unnamed)', category: cat, vramMB: parseFloat((size / (1024 * 1024)).toFixed(2)), width: tex.width, height: tex.height }); }); details.sort(function(a, b) { return b.vramMB - a.vramMB; }); var summaryMB = {}; for (var k in summary) { summaryMB[k] = parseFloat((summary[k] / (1024 * 1024)).toFixed(2)); } return JSON.stringify({ summary: summaryMB, count: details.length, details: details }); }"
```

### 3. 結果を表示

取得したJSONをパースし、以下の2つのテーブルで報告する：

**カテゴリ別サマリー:**

| カテゴリ | 説明 | VRAM (MB) |
|---|---|---|
| asset | 自分で用意した画像 | — |
| shadow | シャドウマップ（影） | — |
| skybox | スカイボックス/環境マップ | — |
| font | UIフォント・テキスト | — |
| other | その他（内部バッファ等） | — |

**テクスチャ詳細（VRAM降順）:**
Texture Name / Category / VRAM (MB) / Width / Height
