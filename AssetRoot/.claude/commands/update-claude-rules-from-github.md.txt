# update-claude-rules-from-github

PlayCanvas Editorが開いている状態で実行すること。
開いていなければ /open-playcanvas-editor を先に実行する。

`<!-- TEMPLATE_BOUNDARY -->` より上（共通テンプレート部分）がリポジトリから更新され、
より下（プロジェクト固有セクション）は保持される。この仕組みは CLAUDE.md だけでなく、
TEMPLATE_BOUNDARY を含む全ファイルに適用される。

## 手順

```bash
playwright-cli eval "() => fetch('https://raw.githubusercontent.com/FrameSynthesis/PlayCanvasEditor_ClaudeCodeRules/main/update.js').then(r=>r.text()).then(t=>{eval(t); return 'started';})"
```

実行結果のログを確認し、エラーがあればユーザーに報告する。
タイムアウトした場合はユーザーにブラウザコンソール(F12)への手動ペーストを案内する。
